핵심 기능

  • 챗봇별 모델/DB/프롬프트 독립 운영
  • Parent→Child 위임 및 응답 종합
  • SSE 실시간 응답 스트리밍
  • 사용자-챗봇 권한 제어

사용 방법

  • 챗봇 선택 후 질문 입력
  • 실시간 답변 확인
  • 필요 시 후속 질문으로 대화 유지
  • 관리자 화면에서 챗봇/권한 운영

도입 효과

  • 답변 리드타임 단축
  • 팀 지식 사일로 해소
  • 운영 자동화/유지보수 단순화
  • 보안/거버넌스 강화

발표용 데모 포인트

  • Child 협업 종합 시나리오
